README : site internet professeur

Début de création d'un site internet destiné à être utilisé par un professeur
du secondaire par exemple. On y trouve  : 

- Page principale : site_prof1 
	Sur cette page on peut choisir, en tant qu'élève, la classe à laquelle on appartient
- Une page secondaire : site_prof_mdp
	Cette page correspond à un click sur la classe 1 et demande le mot de passe afin de pouvoir
	accéder aux cours et aux exercices de la classe correspondante
-Une page secondaire :site_prof2 
	Cette page correspond à la section d'une classe sur laquelle le professeur a chargé des cours
	ou des exercices 

Le mot de passe actuel est : coucou
Lorsqu'on clique sur cours ou exercice on voit apparaître l'attestation de déplacement (exemple)
